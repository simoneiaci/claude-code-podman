# Security Policy

## Supported Versions

Security updates are provided for the latest `main` branch.
If release tags are published, only the most recent tag is supported.

## Reporting a Vulnerability

Please do not open public issues for security reports.

Preferred: GitHub Security Advisories
https://github.com/simoneiaci/claude-code-podman/security/advisories/new

If you cannot use the advisory flow, contact the maintainer via GitHub:
https://github.com/simoneiaci

## Response Timeline

- Acknowledge within 3 business days.
- Provide an initial assessment within 7 business days.
- Aim to ship a fix within 90 days, depending on complexity.

## Scope

In scope:
- The repository source code and documentation.
- The container image built from the Containerfile.

Out of scope:
- Social engineering or phishing attempts.
- Denial of service attacks.
- Vulnerabilities in third-party services outside this repository.
